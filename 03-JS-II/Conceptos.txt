bucle for:
Los bucles for tienen una sintaxis única.
Primero tenemos la palabra clave for, seguida de paréntesis y luego abrir y cerrar llaves.
Dentro de los paréntesis necesitaremos debemos hacer tres pasos.
Primero, declarar una variable, esto es sobre lo que se repetirá el bucle.
Segundo, una expresión condicional, el ciclo continuará sucediendo hasta que esta declaración sea false.
Tercero, incrementaremos nuestra variable. Las tres declaraciones están separadas por un punto y coma.

Operadores lógicos:
los operadores logicos son;
&& se conoce como ("Y").
Está escrito con dos símbolos (&&).
Esto evaluará ambas expresiones y devolverá verdadero si ambas expresiones son true.
Si uno (o ambos) de ellos es falso, este operador devolverá false.

|| se conoce como ("O").
Está escrito con dos barras verticales (||).
Determinará si una de las expresiones es true.
Devolverá true si una (o ambas) de las expresiones es true.
Devolverá false si ambas expresiones son false.

! se conoce como(“NO”). Está escrito como un solo signo de exclamación (!).
Este operador antes al determinar la igualdad (!==).
El operador NO devolverá el valor booleano opuesto de lo que se le pasa.

Nos sirbe para combinar dos expresiones de igualdad y preguntar si alguna de las dos es verdadera,
si ambas son verdaderas o si ninguna de ellas es verdadera.
Para hacer esto, utilizaremos operadores lógicos.
